# 🌿 AI Plant Disease Detector

An AI-powered web application that detects plant diseases from leaf images using Deep Learning.

## 🚀 Live Demo
https://ai-plant-disease-detector.streamlit.app

## 📸 Features
- Upload a plant leaf image
- AI predicts the disease
- Confidence score
- Disease description
- Treatment suggestions
- Heatmap visualization (Grad-CAM)

## 🧠 Tech Stack
- Python
- PyTorch
- Streamlit
- OpenCV
- NumPy
- Computer Vision

## 📂 Project Structure
app.py – Streamlit web app  
predict.py – model prediction  
gradcam.py – heatmap visualization  
disease_info.py – disease description  
model.pth – trained model  

## ⚙️ Run Locally

Clone repository:

git clone https://github.com/aamirhashmi111-byte/AI-Plant-Disease-Detector.git

Install requirements:

pip install -r requirements.txt

Run app:

streamlit run app.py
